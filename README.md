# Student Data Analysis System

This is a Python-based project that analyzes student performance data.

## 📌 Features
- Accepts marks of multiple students
- Calculates average, highest, and lowest marks
- Assigns grades (A, B, C, Fail)
- Displays pass/fail status for each student

## 🛠 Technologies Used
- Python

## ▶️ How to Run
1. Open the file in Python IDLE
2. Run the program (F5)
3. Enter number of students
4. Enter marks when prompted
5. View the analysis result

## 📊 Output Example
- Average Marks
- Highest Marks
- Lowest Marks
- Grade and Status for each student
